## 文件清单

```
server.js                    # Express 后端服务
database/store.js            # JSON 数据存储层（替代 sql.js）
database/data.json           # 数据文件（自动生成）
database/init.js             # 旧版数据库初始化（保留但未使用）
public/
  index.html                 # SPA 入口
  css/style.css              # 全局样式（专业教育风格）
  js/
    api.js                   # API 封装层
    app.js                   # 路由 + 全局工具函数
    modules/
      portal.js              # 门户首页（轮播图/课程/AI问答/证书查询）
      student.js             # 学员平台（学习/考试/证书/个人中心）
      teacher.js             # 教师平台（班级/成绩/直播/公告）
      admin.js               # 管理后台（教学/资源/财务/组织管理）
      dashboard.js           # 数据看板（Chart.js 图表）
      settings.js            # 基础数据管理（10项全部实现）
```
